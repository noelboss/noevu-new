# Blog Post Template Structure

This document describes the common structure used across all blog posts on noevu.ch.

---

## Common Blog Post Structure

### Header Section
**Type:** `blogHeader`
**Theme:** `white`

**Content:**
- Category tag/link
- Headline (H1): Post title
- Subheading: Post excerpt/summary
- Meta info: Author, date, reading time
- Featured image (full-width or contained)

---

### Featured Image Section
**Type:** `featuredImage`
**Theme:** `white`
**Layout:** Full-bleed or contained

**Content:**
- Hero image for the post
- Optional caption
- Alt text for SEO

---

### Content Section
**Type:** `richText`
**Theme:** `white`
**Layout:** Narrow content width (readable)

**Content:**
- Markdown/rich text content
- Headings (H2, H3, H4)
- Paragraphs
- Lists (ordered/unordered)
- Blockquotes
- Code blocks (if technical)
- Inline images
- Internal/external links

---

### Embedded Tool Section (Optional)
**Type:** `embedSection`
**Theme:** `light`

**Content:**
- Fillout.com embeds (e.g., CMS Check tool)
- Cal.com booking widgets
- Other third-party embeds

**Example (CMS Check):**
- Fillout form embed
- Custom styling/branding

---

### Image Gallery Section (Optional)
**Type:** `imageGallery`
**Theme:** `white`
**Layout:** Grid or masonry

**Content:**
- Multiple images
- Lightbox functionality
- Captions

---

### FAQ/Accordion Section (Optional)
**Type:** `faqAccordion`
**Theme:** `light-bold`

**Content:**
- Related questions and answers
- Expandable/collapsible items

---

### Author Box Section
**Type:** `authorBox`
**Theme:** `white`
**Layout:** Card with image

**Content:**
- Author photo (circular)
- Author name
- Author title/role
- Short bio
- Social links (optional)

---

### Related Posts Section
**Type:** `relatedPosts`
**Theme:** `light`
**Layout:** 3-column grid

**Content:**
- 3 related blog posts
- Post thumbnails
- Post titles
- Category tags

---

### CTA Section
**Type:** `ctaSection`
**Theme:** `dark` or `light-bold`
**Background:** Optional generative animation

**Content:**
- Call to action headline
- CTA button
- Contact information

---

### Navigation Section
**Type:** `postNavigation`
**Theme:** `white`

**Content:**
- Previous post link
- Next post link
- Back to blog link

---

### Footer
**Type:** `footer`
**Theme:** `dark`

---

## Blog Post Categories

1. **Portfolio** - Case studies and project showcases
2. **Guide** - Tutorials and how-to articles
3. **CMS** - CMS comparison and technology articles
4. **AI** - Artificial intelligence and automation articles
5. **Tools** - Software tools and productivity articles

---

## Common Components Required
- Blog header with meta
- Featured image display
- Rich text renderer
- Embed container (Fillout, Cal.com)
- Image gallery
- FAQ accordion
- Author box card
- Related posts grid
- Post navigation
- Footer

---

## SEO Considerations
- Structured data (Article schema)
- Open Graph meta tags
- Twitter Card meta tags
- Canonical URLs
- Reading time calculation
- Table of contents (optional)
