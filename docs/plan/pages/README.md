## URL Structure

### German (Default)

```
noevu.ch/                           → Homepage
noevu.ch/[page-slug]                → Static pages
noevu.ch/blog                       → Blog listing
noevu.ch/blog/[post-slug]           → Blog post
noevu.ch/blog/category/[cat-slug]   → Category page
noevu.ch/blog/tag/[tag-slug]        → Tag page
```

### English

```
noevu.ch/en/                        → Homepage (EN)
noevu.ch/en/[page-slug]             → Static pages (EN)
noevu.ch/en/blog                    → Blog listing (EN)
noevu.ch/en/blog/[post-slug]        → Blog post (EN)
```
